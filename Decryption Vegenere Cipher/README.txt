Algoritmo de decriptação cifra de Vigenere
--------------------------------------------------------------------------------------------------------------

para utiliza-lo você deve utilizar o comando em seu terminal:
./vigenere "Arquivo contendo a mensagem a ser decriptada" "senha de decriptação"

importante: o alfabeto para esse algoritmo está padronizado para o alfabeto "abcdefghijklmnopqrstuvxwyz"

exemplo de execução:
./vigenere cifra.txt senha

Decryption Algorithm Vigenere Cipher
--------------------------------------------------------------------------------------------------------------

To use you must use this command in your terminal:
./vigenere "File with the message that will be decrypted" "decryption password"

important: the alphabet used for this algorithm is "abcdefghijklmnopqrstuvxwyz"

execution exemplo:
./vigenere cipher.txt password
